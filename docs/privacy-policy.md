# 妙记隐私政策

更新日期：2026 年 7 月 17 日
生效日期：首次正式发布之日

妙记由个人开发者 **张静**（以下称“我们”）运营。我们重视你的个人信息和选择权。本政策说明妙记在本地手动记账、可选云同步和可选语音记账场景下如何处理信息。

如果你不同意云同步及跨境传输，可以不登录，继续使用本地手动记账、统计和 CSV 导出功能。

## 1. 我们处理的信息

| 场景 | 信息类型 | 目的 | 是否必要 |
| --- | --- | --- | --- |
| 本地手动记账 | 收支金额、标题、备注、分类、日期、预算、货币和应用偏好 | 在设备上记录、展示和统计账目 | 手动记账所必需，仅保存在本机时不会上传 |
| 云同步登录 | 邮箱地址、Supabase 用户标识、登录会话 | 创建账号、登录、鉴权和恢复账号 | 云同步所必需，可拒绝 |
| 云同步 | 完整账本快照，包括上述账目、分类、预算和偏好 | 跨设备同步和恢复 | 云同步所必需，可拒绝 |
| 语音记账 | 你主动录制并提交的音频、当前分类名称及标识 | 将语音解析为待确认的记账草稿 | 语音记账所必需，可拒绝麦克风权限或不使用该功能 |
| 用户支持 | 你主动发送的邮箱地址、问题描述、截图等 | 回复问题、排查故障 | 可选 |

请不要在账目、语音或支持邮件中提交与记账无关的身份证号码、完整银行卡号、密码、验证码或他人的个人信息。

我们不接入广告 SDK，不跨应用跟踪用户，不出售个人信息，也不会把账本或录音用于广告画像。

## 2. 本地处理与权限

- 本地账本默认存储在你的 iPhone 上。
- 麦克风权限只在你主动使用语音记账时请求；拒绝后仍可手动记账。
- 本地登录令牌存储在 iOS 钥匙串。
- CSV 文件仅在你主动导出时生成，导出后的保管和分享由你控制。

## 3. 委托处理和第三方服务

为提供在线功能，我们使用以下服务商，并只提供实现相应功能所需的数据：

1. **Supabase Pte. Ltd（新加坡）**：提供邮箱登录、账号鉴权、账本快照数据库及语音文件临时存储。处理的信息包括邮箱、用户标识、登录信息、账本快照和用户主动提交的录音。其[隐私政策](https://supabase.com/privacy)和[数据处理附录](https://supabase.com/downloads/docs/Supabase%2BDPA%2B260601.pdf)适用。联系地址：65 Chulia Street #38-02/03, OCBC Centre, Singapore 049513；隐私联系邮箱：privacy@supabase.io。
2. **通义云启（杭州）信息技术有限公司（阿里云百炼，中国大陆）**：对用户主动提交的录音进行语音理解并生成结构化记账草稿。处理的信息包括录音内容及当前分类名称，不包含你的邮箱和完整账本。其[阿里云百炼隐私政策](https://terms.alicdn.com/legal-agreement/terms/privacy_policy_full/20230801173543354/20230801173543354.html)适用。
3. **腾讯云（中国大陆，北京，计划中的正式服务部署）**：运行妙记 API，负责鉴权、接收录音、协调临时存储和调用百炼。正式发布前将以实际购买和备案完成的资源为准更新本项。
4. **Apple**：提供 App Store 分发、系统权限、崩溃或诊断能力；Apple 按其政策处理相关信息。

第三方服务商可能依法使用其分包处理方。我们会依据服务协议、安全能力和最小必要原则选择服务商；服务商或处理地区发生实质变化时，我们会更新本政策，并在依法需要时重新征求同意。

## 4. 个人信息跨境传输

当你主动开启云同步或语音记账时，邮箱、账号标识、账本快照以及你主动提交的录音会传输至位于**新加坡**的 Supabase 项目。目的分别是账号鉴权、账本同步与恢复、录音临时存储；传输方式为加密的 HTTPS 网络连接。录音随后通过限时访问地址交由中国大陆的阿里云百炼解析。

在首次发送验证码或使用邮箱密码登录前，应用会通过独立开关征求你的**单独同意**。你可以拒绝，此时仍可使用本地手动记账，但不能使用云同步和语音记账。你也可以退出账号或在应用内永久删除账号和关联数据，撤回后不影响此前基于同意进行处理的效力。

境外接收方为 Supabase Pte. Ltd，其名称、地址、联系方式、处理目的、信息类型和处理方式见本政策第 3 节。你可以通过本政策第 7 节联系我们，或通过上述 Supabase 隐私邮箱，就境外处理行使查询、更正、复制、删除等权利。

> 运营合规说明：正式发布前，运营者仍需根据中国个人信息跨境规则完成个人信息保护影响评估、留存单独同意记录，并确认所适用的跨境传输条件或豁免。本段不是对已经完成全部法律手续的声明。

## 5. 保存期限和安全措施

- 本地账本保存到你主动清除、删除 App 或永久删除账号为止。
- 云端账号和账本保存到你在应用内永久删除账号为止；法律法规另有保存要求的除外。
- 录音上传后通常会在每次解析尝试结束时立即从 Supabase 删除。正式发布前还会配置兜底任务，清理超过 24 小时的孤立录音；本机录音在成功解析或取消时删除，并清理超过 24 小时的缓存。
- 我们使用 HTTPS、私有存储桶、限时签名链接、账号身份验证、数据库行级安全策略和最小权限密钥等措施保护数据。仅服务端持有 Supabase Secret Key 和百炼 API Key。

任何网络服务都无法保证绝对安全。如发生可能影响你权益的个人信息安全事件，我们会依法采取补救措施并履行通知义务。

## 6. 你的权利

你可以在应用内：

- 查看、更正或删除账目；
- 导出 CSV，或清除本机全部记账记录；
- 退出云同步账号；
- 通过“设置 → 云同步 → 永久删除账号与数据”删除云同步账号、云端账本和本机账本。

如需查询、复制、更正、补充、删除个人信息，撤回同意，注销账号，或对本政策提出疑问和投诉，请通过第 7 节联系我们。我们将在核验身份后于法律要求的期限内处理。

我们可能因功能、部署地区、服务商或法律要求更新本政策。发生处理目的、处理方式、信息种类或跨境接收方等重大变化时，我们会通过应用内提示、版本说明或其他适当方式告知，并在依法需要时重新征求同意。
